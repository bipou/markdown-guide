# Markdown语法和实例

## 一、标题

Markdown中共有6级标题，从 `#`、`##`、依次类推到 `######`。

### 1. 实例代码

``` Markdown
# 一级标题
## 二级级标题
### 三级标题
#### 四级级标题
##### 五级标题
###### 六级标题
```

### 2. 呈现效果

<div style="border:1px solid;margin:0px;padding:0px 0px 0px 15px">

# 一级标题

## 二级级标题

### 三级标题

#### 四级级标题

##### 五级标题

###### 六级标题

</div>

## 二、字体

用 `*` 或者 `_` 包裹文本进行 _斜体_ 格式化，用 `**` 或者 `__` 包裹文本进行 **粗体** 格式化。

`**` 或者 `__` 可以和  `*` 或者 `_` **_组合使用_**。

### 1. 实例代码

``` Markdown
*这行文本是斜体*
_这行文本是斜体_

**这行文本是粗体**
__这行文本是粗体__

_这行文本是 **粗体** + 斜体_
```

### 2. 呈现效果

<div style="border:1px solid;margin:0px;padding:0px 0px 0px 15px">

*这行文本是斜体*

_这行文本是斜体_

**这行文本是粗体**

__这行文本是粗体__

_这行文本是 **粗体** + 斜体_

</div>
