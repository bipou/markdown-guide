Markdown和HTML融合

Markdown中，可以直接嵌入HTML内容，包括：tags，css等。

嵌入HTML-tags

<div style="border:1px solid;margin:0px;padding:0px 0px 0px 15px">

嵌入CSS

1. 实例代码

<div style="border:1px solid;margin:0px;padding:0px 0px 0px 15px">

<style type="text/css">
    .indent-background {
        text-indent:2em;
        background: red;
    }
</style>

<div class="indent-background" style="margin:10px">
